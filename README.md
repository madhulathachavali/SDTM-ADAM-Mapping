# SDTM-ADAM-Mapping

This analysis investigates the systolic blood pressure (SYSBP) trends across different visits and treatment groups.

### Steps

1. Load raw data

2. Mapping to SDTM domains:

- DM (Demographics)

- VS (Vital Signs)

- AE (Adverse Events)

3. Mapping to ADaM datasets:

- ADSL (Subject-Level)

- ADVS (Vital Signs Analysis)
